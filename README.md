# AI Resume Analyzer & Interview Coach

An AI-powered application that analyzes resumes and helps users prepare for interviews.

Features

- Resume analysis using AI
- Skill gap detection
- Interview question generation
- AI-based answer evaluation

Tech Stack

Python ,
Streamlit ,
Groq ,
PyPDF2 ,
Fpdf .

Installation

1. Clone repository

git clone https://github.com/Vamshi-2404/AI-Resume-Analyzer-Interview-Coach.git

2. Install dependencies

pip install -r requirements.txt

3. Add Groq API key

export GROQ_API_KEY=your_api_key

4. Run app

streamlit run app.py
